# メモアプリ 設計書

作成日: 2026-05-17

## 概要

ブラウザで動くシンプルなメモアプリ。
サーバー不要、インストール不要。`index.html` をブラウザで開くだけで動く。

## 機能

| 機能 | 状態 |
|------|------|
| 新規メモの作成 | ✅ 完成 |
| メモの一覧表示（左サイドバー） | ✅ 完成 |
| タイトルと本文の編集（右エリア） | ✅ 完成 |
| データの自動保存（localStorage） | ✅ 完成 |
| メモの削除 | ✅ 完成 |
| Undo機能 | 🔲 未実装 |
| レスポンシブデザイン（モバイル対応） | 🔲 未実装 |
| ミニマルデザイン | 🔲 未実装 |

## UI 構成

### デスクトップ（600px以上）

| エリア | 内容 |
|--------|------|
| 左ペイン | 新規メモボタン、メモ一覧（タイトル・更新日）、削除ボタン |
| 右ペイン | タイトル入力欄、本文テキストエリア |

### モバイル（600px未満）

| 画面 | 内容 |
|------|------|
| リスト画面 | メモ一覧のみ表示 |
| 編集画面 | 編集エリアのみ表示、「← 戻る」ボタンあり |

## Undo機能

- `Ctrl+Z` で直前の操作を取り消す
- 「UNDO」ボタンで直前の操作を取り消す
- 対象：メモの作成・削除・編集
- 実装方法：操作前の状態を履歴スタックに保持する
- 最大履歴数：20件

## ミニマルデザイン方針

- 余白を広めにとり、スッキリした印象に
- ボーダーや区切り線を最小限にする
- サイドバーの背景色を柔らかくする
- ボタンをシンプルなスタイルに統一

## 技術スタック

| 技術 | 用途 |
|------|------|
| HTML | 画面の構造 |
| CSS | スタイル（純粋なCSS、外部ライブラリなし） |
| JavaScript | ロジック（バニラJS、フレームワークなし） |
| localStorage | データの永続化（ブラウザ内に保存） |
